# 📊 Sales Data Analysis Dashboard | Microsoft Excel

## 📌 Project Overview

This project demonstrates an end-to-end sales data analysis workflow using **Microsoft Excel**. Starting with raw sales transaction data, the dataset was cleaned, transformed, and organized into a master table before being analyzed using Pivot Tables and visualized through an interactive dashboard.

The dashboard provides valuable business insights into sales performance, profitability, customer behavior, product performance, and regional trends, helping support data-driven decision-making.

---

## 🎯 Objectives

- Clean and prepare raw sales data.
- Create a structured master table for analysis.
- Analyze sales performance using Pivot Tables.
- Build an interactive dashboard.
- Present business KPIs and actionable insights.

## 🧹 Data Cleaning & Preparation

The raw dataset was cleaned and prepared in Microsoft Excel before analysis.

### Data Cleaning

- Imported the raw sales dataset.
- Converted the data into an Excel Table.
- Removed duplicate records.
- Verified missing and inconsistent values.
- Standardized date formats.
- Corrected data types (Date, Currency, Number, Percentage).
- Cleaned text fields by removing extra spaces.
- Standardized text formatting for consistency.


### Data Preparation

Tasks performed:

- Converted the dataset into an Excel Table.
- Verified data consistency and formatting.
- Standardized date, currency, and percentage formats.
- Created calculated columns:
  - Gross Amount
  - Sales
  - Cost
  - Profit
- Extracted Month and Year from Order Date for time-based analysis.
- Organized the cleaned data into a Master Table for reporting.
